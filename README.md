☁Quiz Cloud-Interactive Quiz Platform

Codsoft Webdevelopment Internship-Task4

📝Project Description:

*QuizCloud is a robust, full-feature web application that allows users to create, manage, and take interactive quizzes. Developed as the final task for my CodSoft Internship, this project demonstrates advanced JavaScript logic, state management, and data persistence without a backend database.

*The app provides a seamless flow from a "Creator Mode" where users can input custom questions, to a "Player Mode" featuring a real-time countdown timer and instant performance analytics.

✨Features:

*Dynamic Quiz Creator: Add as many questions and options as needed with a user-friendly form.

*Smart Quiz Engine: Automatic question shuffling and instant feedback (Green/Red) on answers.

*Persistent Data: All created quizzes and user scores are saved to LocalStorage, ensuring no data is lost on page refresh.

*Timed Challenges: Integrated countdown timer per quiz to simulate real exam conditions.

*Result Analytics: Detailed summary page showing total score, total questions, and accuracy percentage.

*Glassmorphic Dashboard: A modern, high-end UI featuring a sidebar navigation and frosted glass cards.

🛠Tech Stacks:

*HTML
*CSS
*JAVASCRIPT
